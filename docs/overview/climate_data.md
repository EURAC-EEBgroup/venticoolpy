### Climate data files 

To perform the calculation, you have to provide the path to a file containing hourly meteorological data for a full year.

Accepted file formats: .csv or .epw 

Possible data sources:  

- Repository of Building Simulation Climate Data: [https://climate.onebuilding.org/](https://climate.onebuilding.org/)  
- Joint Research Centre, “TMY generator,” 2022. [https://joint-research-centre.ec.europa.eu/pvgis-online-tool/pvgis-tools/tmy-generator_en](https://joint-research-centre.ec.europa.eu/pvgis-online-tool/pvgis-tools/tmy-generator_en) 

The calculation should be also performed with future weather datasets if available to ensure resilience of ventilative cooling strategies over time.
